# iOS App Setup

Since this project was initialized on Windows, the Xcode project file (`.xcodeproj`) could not be generated instantly.

## How to run on Mac
1. Open Xcode.
2. Create a new iOS App project named `iosApp` in this directory.
3. Replace the generated `ContentView.swift` and `iOSApp.swift` with the files in `iosApp/`.
4. Ensure the Build Settings in Xcode link against the `ComposeApp.framework` generated by Gradle.
   - Run `./gradlew :composeApp:embedAndSignAppleFrameworkForXcode` (or similar task depending on configuration).
   - Add the framework search path `$(SRCROOT)/../composeApp/build/xcode-frameworks/$(CONFIGURATION)/$(SDK_NAME)`.

## Files Provided
- `iOSApp.swift`: The main entry point.
- `ContentView.swift`: Bridges the Kotlin `MainViewController` to SwiftUI.
